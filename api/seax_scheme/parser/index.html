Module seax_scheme::parser [-] [+] [src]

Contains the Scheme parser.

This parser is based on the Scheme grammar given in the Revised6 Report on Scheme (R6RS). Any deviations from the R6RS standard, especially those with an impact on the valid programs accepted by the parser, will be noted in the parser's RustDoc.
